# Deep_Web_Crawler

This is a command line search tool allows a user to enter a search term or phrase in the command line to pull the top deep web sites relating to that query using the darksearch.io API.

<h2> Installation: </h2>
<ol>
 <li> Download/Clone the repository </li>
 <li> cd ./Deep_Web_Crawler-main/Deep_Web_Crawler-main/src </li>
 <li> javac DWeb_Crawler/Runner.java </li>
 <li> java DWebCrawler/Runner </li>
</ol>
